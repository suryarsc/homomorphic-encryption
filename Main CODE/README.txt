Note: Two different instances of encrypted form of same number (x) with same public key (n) doesn't need to be same.


Note: Paillier comes under Partially Homomorphic Encryption (PHE)
Note: Paillier is designed to work only on integers
Note: Paillier supports operations like addition, substraction, and scalar multiplication

Note: BFV comes under Fully Homomorphic Encryption (FHE)
Note: BFV is designed to work only on integers
Note: BFV supports operations like addition, substraction, and multiplication

Note: CKKS comes under Fully Homomorphic Encryption (FHE)
Note: CKKS is designed to work only on floating-point nubmers
Note: CKKS supports operations like addition, substraction, and multiplication
